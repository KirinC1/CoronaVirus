Is the Coronavirus a Serious Pandemic?

I think that this pandemic is definitely NOT being blown out of proportion. I think this because this is something that if we do not try to contain it, it will spread very fast and become way more deadly than we think it is now. I think that we need to keep a serious mindset about it, but, there is no need to panic. I think that there are a lot of people around the world that are panicking way too much. If we keep calm, stay healthy, clean, and try our best not to spread it to others if we possibly have it, then I think that this virus will be beaten quickly.
